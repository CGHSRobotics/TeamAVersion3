# TeamAVersion3

## Motor Setup

    Port 1 = Front Left Drive
    Port 12 = Back Left
    Port 3 = Front Right
    Port 14 = Back Right
    Port 5 = Conveyor Motor
    Port 6 = Intake Motor
    Port 7 = Launcher Motor
    Port 8 = undefined
